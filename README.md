# ALHR Algorithmic Hairline Reconstruction

A procedural generation tool for hair transplant planning and hairline design. This project uses Poisson Disk Sampling and Bezier curves to simulate natural hair follicle distributions.

## Project Status Work in Progress
Note: This project is currently under active development and is not yet complete. The current version v0.1 is a functional prototype, but many planned features and optimizations are still pending.

## Features
* Procedural Distribution: Uses Poisson Disk Sampling for organic, non overlapping graft placement.
* Dynamic Arc Modeling: Define hairlines using interactive anchor points and Bezier curvature.
* Natural Transitioning: Stochastic feathering logic to simulate low density leading edges.
* Interactive Controls: Real time adjustment of spacing, depth, and density.

## Getting Started
Currently, this is a static web application.
1. Clone the repository.
2. Open alhr.html in any modern web browser.
